Computer

Definitions

  • Computer: An electronic machine capable of storing information and processing it automatically through mathematical and logical operations controlled by software programs.

  • Compatible computer: A computer capable of running the same programs as other computers of the same type.

  • Personal computer: A desktop computer that includes a central unit, keyboard, screen, and one or more disk drives, and can function without being connected to a computer network.

  • Laptop: A personal computer that is small and lightweight enough to be easily carried around, like a briefcase; a built-in battery provides some operational autonomy.
